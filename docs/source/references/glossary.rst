Glossary
========

This page includes a number of terms that we use in our documentation.

.. glossary::

   AWS
      Amazon Web Services

      https://aws.amazon.com/

   AWS CLI
      Amazon Web Services Command Line Interface is a unified tools for interacting with all parts of AWS

      https://aws.amazon.com/cli/

   github home
      https://github.com/orgs/RCIC-UCI-Public/rcs3

   IAM Roles
      AWS Identity and Access Management roles

      https://docs.aws.amazon.com/IAM/latest/UserGuide/introduction.html

   Lifecycle rules
      Is a set of rules that define the actions applied by S3 to a group of objects.

      https://docs.aws.amazon.com/AmazonS3/latest/userguide/lifecycle-transition-general-considerations.html

   MFA
      Multi-factor authentication

   project home
     https://rcs3.readthedocs.io

   Rclone
      Rclone is a command-line program to manage files on cloud storage.
      It is a feature-rich alternative to cloud vendors' web storage interfaces.

      https://rclone.org/

   RCIC
      Research Cyberinfrastructure Center at the University of California, Irvine.

      https://rcic.uci.edu/

   RCS3
      Rclone to S3 for large backup

      https://rcs3.readthedocs.io

   S3
      Simple Storage Service

      See https://docs.aws.amazon.com/s3

   S3 Versioning
      S3 Versioning allows keeping multiple versions of an object in one bucket,
      so that you can restore objects that are accidentally deleted or overwritten.

      See https://docs.aws.amazon.com/AmazonS3/latest/userguide/Versioning.html

   UCI
      University of California, Irvine

      https://uci.edu/
